Objective
To design and develop a secure and user-friendly ATM Interface that allows users to perform basic banking operations such as login authentication, cash withdrawal, deposit, fund transfer, balance enquiry, and transaction history using Java Swing.
________________________________________
Features
•	Secure Login Authentication (User ID & PIN)
•	Withdraw Money with balance validation
•	Deposit Money
•	Fund Transfer
•	Balance Enquiry
•	Transaction History
•	Exit Functionality
•	Attractive and responsive GUI using Java Swing
•	Error handling for invalid inputs
________________________________________
Outcome
•	Successfully developed a functional ATM simulation system.
•	Improved understanding of Java Swing components and event handling.
•	Gained hands-on experience in GUI-based application development.
•	Learned to manage user data and transactions using Java collections.
•	Enhanced problem-solving and debugging skills.
